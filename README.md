# 🎮 Тетрис - Консольная игра на C++

<div align="center">


</div>

Это реализация классической игры **Тетрис** для терминала Linux/Unix. Игра поддерживает несколько режимов и предлагает современный игровой опыт в консольной среде.

## ✨ Особенности

| Функция | Описание |
|---------|----------|
| 🎨 **Цветная графика** | Полноцветное отображение в терминале |
| ⚙️ **Настраиваемое управление** | Возможность переназначения всех клавиш |
| 🏆 **Система рекордов** | Сохранение лучших результатов в файл |
| ⏸️ **Режим паузы** | Временная остановка игры |
| 🎲 **Разные режимы** | Несколько вариантов игры |

## 🎯 Режимы игры

### 1. 🏛️ Классический Тетрис
### 2. 🪣 Тетрис в форме ведра
### 3. 🧩 Режим "Собери картинку"
### 🎲 Режимы последовательности фигур:
```bash
# Без аргументов - стандартный режим (случайные фигуры)
./tetris_game

# С аргументом "1" - только квадраты (фигуры O)
./tetris_game 1

# С аргументом "2" - чередование фигур по порядку
./tetris_game 2
```

## 🎮 Управление

### ⌨️ Стандартные клавиши:

| Клавиша | Действие | Альтернатива |
|---------|----------|--------------|
| **A** / **←** | Движение влево | Стрелка влево |
| **D** / **→** | Движение вправо | Стрелка вправо |
| **S** / **↓** | Ускоренное падение | Стрелка вниз |
| **W** / **↑** | Мгновенный сброс | Стрелка вверх |
| **Пробел** | Поворот фигуры | |
| **E** | Пауза / Продолжить | |
| **Q** | Выход из игры | |

> **💡 Все клавиши можно переназначить в настройках игры**

## ⚙️ Установка

### Шаги установки:

```basht
cd tetris
mkdir build
cd build
cmake ..
make
```

## 🚀 Запуск

```bash
# Запуск игры
./tetris_game

# Или с параметрами:
./tetris_game 1    # Режим только квадраты
./tetris_game 2    # Режим чередования фигур
```
