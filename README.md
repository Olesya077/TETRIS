### 2. Тетрис в форме ведра

### Режимы последовательности фигур:
Игра поддерживает числовой аргумент для установки зерна рандомизации (seed). Это позволяет воспроизводить одинаковую последовательность фигур при разных запусках.
```bash
# Без аргументов - используется текущее время как seed (случайная последовательность)
./tetris_game

# С числовым аргументом - фиксированный seed для воспроизводимой последовательности
./tetris_game 42        # Всегда одинаковая последовательность с seed 42
./tetris_game seed # Другой фиксированный seed
```

## Управление

### Стандартные клавиши:

| Клавиша | Действие | Альтернатива |
|---------|----------|--------------|
| **A** / **←** | Движение влево | Стрелка влево |
| **D** / **→** | Движение вправо | Стрелка вправо |
| **S** / **↓** | Ускоренное падение | Стрелка вниз |
| **W** / **↑** | Мгновенный сброс | Стрелка вверх |
| **Пробел** | Поворот фигуры | |
| **E** | Пауза / Продолжить | |
| **Q** | Выход из игры | |

> **Все клавиши можно переназначить в настройках игры**
