Подробное описание

Класс для обработки ввода с терминала в неблокирующем режиме

Настраивает терминал для чтения одиночных символов без эха и ожидания Enter. Восстанавливает оригинальные настройки терминала при уничтожении.

Конструктор - настраивает терминал

Заметки
Отключает:
  • Канонический режим (ожидание Enter)

12 {
13 /**
14 * @brief Конструктор - настраивает терминал
15 * @note Отключает:
16 * - Канонический режим (ожидание Enter)
17 * - Эхо ввода
18 * Устанавливает:
19 * - Чтение по 1 символу
20 * - Без таймаута
21 */
22 struct termios term;
23 tcgetattr(STDIN_FILENO, &originalTerm);
24 term = originalTerm;
25 term.c_lflag &= ~(ICANON | ECHO);
26 term.c_cc[VMIN] = 1;
27 term.c_cc[VTIME] = 0;
28 tcsetattr(STDIN_FILENO, TCSANOW, &term);
29}
