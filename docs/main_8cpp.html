Точка входа в программу

Инициализирует генератор случайных чисел и запускает игровой контроллер. Поддерживает аргумент командной строки для установки зерна рандомизации.

Аргументы
argcКоличество аргументов командной строки
argvМассив аргументов командной строки
Возвращает
0 при успешном завершении

Аргументы командной строки

Игра поддерживает один опциональный числовой аргумент, который используется в качестве зерна для генератора псевдослучайных чисел. Это позволяет воспроизводить одинаковую последовательность фигур при разных запусках.

Примеры использования:

  • ./tetris_game # Использует текущее время как seed
  • ./tetris_game 42 # Использует 42 как seed (фиксированная последовательность)
  • ./tetris_game 123456789 # Другой фиксированный seed
Заметки
Если аргумент не является числом, он будет проигнорирован и использовано текущее время в качестве seed.

Инициализация генератора случайных чисел

Генератор случайных чисел инициализируется одним из двух способов:

  1. Если передан аргумент командной строки: используется как seed
  2. Если аргументов нет: используется текущее время (time(NULL))

Это обеспечивает либо воспроизводимость (с фиксированным seed), либо случайность (с seed от времени) в разных игровых сессиях.
