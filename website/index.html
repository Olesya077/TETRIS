Тетрис

Классическая игра в терминале

Играй в Тетрис прямо в терминале

Проект написан на C++ с использованием объектно-ориентированного программирования. Поддержка цветной графики, рекордов и нескольких режимов игры.

Особенности проекта

Несколько режимов

Классический Тетрис, игра в форме ведра и режим "Собери картинку".

Цветная графика

Яркое оформление в терминале с использованием ANSI-кодов.

Настройка управления

Все клавиши можно переназначить под свои предпочтения.

Таблица рекордов

Сохраняйте свои достижения в файл и соревнуйтесь с другими.

Установка и запуск

# Клонировать репозиторий
git clone https://github.com/Olesya077/TETRIS.git
cd TETRIS

# Собрать проект
mkdir build && cd build
cmake ..
make

# Запустить игру
./tetris_game
